## Pourquoi Astro ?

J'ai commencé à apprendre **Astro** parce que je cherchais un outil simple
pour créer un site web. Venant du HTML et CSS classique, Astro m'a plu
immédiatement parce que la syntaxe reste très proche de ce que je connaissais.

## Ce que j'ai appris jusqu'ici

Voici les concepts que je maîtrise déjà :

- Les **layouts** pour ne pas répéter le HTML commun
- Les **composants** pour réutiliser des morceaux d'interface
- Les **props** pour passer des données entre fichiers
- Le **rendu conditionnel** avec `&&`
- Les **boucles** avec `.map()`
